# Le CISO qui voyait des failles partout

Tu viens d'être nommé CISO d'une scale-up de 200 personnes qui n'a jamais eu de responsable sécurité. Ton prédécesseur ? Il n'existe pas. La "politique de sécurité" ? Un Google Doc de 2019 intitulé "Règles sécu (brouillon)" que personne n'a jamais ouvert. Le mot de passe admin de la prod est "Startup2021!". Il est dans un channel Slack public.

Tu as 15 ans d'expérience en cybersécurité, une certification CISSP, et un ulcère naissant. Ta mission : sécuriser un château de cartes sans le faire tomber. Tes ennemis : les développeurs, les commerciaux, le CEO, le shadow IT, et la nature humaine en général.
